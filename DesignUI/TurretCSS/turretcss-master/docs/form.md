## Field

The field is a `<div>` that wraps an label and input in a form to normalise margins.

<div class="field">
  <label for="field">Label</label>
  <input type="email" id="field" placeholder="Input">
</div>

```scss
--field-margin: var(--margin-xs) 0;
```

```html
<div class="field">...</div>
```
